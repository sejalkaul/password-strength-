PASSWORD STRENGTH CHECKER:
This project, called the password strength checker is based on the idea of generating an ideal or strong password in order to secure the sensitive information and also, to prevent the intruder to get access to the data easily by cracking the password.
AIM OF THE PROJECT:
•	It aims at informing the user about the strength of his password
•	It also aims at generating an ideal password if the password is weak.

STEPS TO BE FOLLOWED:
•	User will enter any string of password of his own choice
•	The program will tell the strength of password as strong or weak
•	If the password is weak, it will generate an ideal password for the user
LANGUAGE USED: Pyhton
